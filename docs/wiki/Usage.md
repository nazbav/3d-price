# Руководство пользователя

1. Откройте `test.html` в браузере или перейдите по ссылке из README.
2. Заполните справочники во вкладках **Принтеры**, **Общие материалы** и **Общие доп. расходы**.
3. На вкладке **Калькулятор** выберите принтер, материал и укажите время печати.
4. Нажмите **Рассчитать** чтобы увидеть итоговую стоимость. Доступна печать этикетки и сохранение в историю.
5. Во вкладке **История** можно фильтровать расчёты по дате и клиенту.
6. Раздел **Клиенты** позволяет хранить контакты и быстро выбирать их при расчётах.
7. В **Настройках** задаются ставки оператора, стоимость электроэнергии, налог, параметры этикетки и данные для облачной синхронизации.
8. Для массового удаления и изменения элементов используйте одноимённые кнопки на страницах списков. Интерфейс автоматически пересчитывает сводные данные.
9. При загрузке страницы, если в истории встречаются расчёты с отсутствующими принтерами, появляется окно для выбора действующего устройства.

Импорт и экспорт данных выполняются через JSON‑файлы или сервис Supabase. При синхронизации создаётся уникальный `sync_id` и QR‑код для мобильных устройств.
