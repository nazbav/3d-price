# Инструкции для участников
0. Все изменения фиксировать в соответствующих файлах `CHANGELOG.md`, `FEATURE_LIST.md`, `TEST_CHANGE_LOG.MD`.
1. После изменений в Python‑скриптах запускайте проверку:
   ```bash
   python -m py_compile open_calc.py sync_orca.py
   ```
2. Документация хранится в каталоге `docs/wiki`. При изменениях в `test.html` обновляйте соответствующие файлы wiki.
3. Перед коммитом убедитесь, что калькулятор работает без ошибок в консольной среде Ubuntu.
   - Запустите локальный сервер в корне проекта:
     ```bash
     python3 -m http.server
     ```
   - Откройте `http://localhost:8000/test.html` в браузере. Без GUI используйте:
     ```bash
     chromium-browser --headless --disable-gpu http://localhost:8000/test.html
     ```
   - Проверьте вывод на отсутствие сообщений `Uncaught` и выполните несколько расчётов.
   - Дополнительно запустите `python open_calc.py sample.gcode` и убедитесь, что ссылка открывается.
4. Используйте Markdown в кодировке UTF‑8. Файлы руководств начинаются с заголовка первого уровня.
