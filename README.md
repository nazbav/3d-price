# 3D-Price Calculator

Этот репозиторий содержит **два** варианта калькулятора стоимости 3D-печати, написанных на чистом JavaScript с использованием Bootstrap 5 для оформления. Приложения работают прямо в браузере и хранят изменения в **LocalStorage**.

- **[Первая версия](https://nazbav.github.io/3d-price/index.html)** (index.html)
  - Базовый калькулятор с учётом:
    - Принтеров (их стоимости и планируемых часов окупаемости),
    - Материалов (тип, цена за кг, наличие),
    - Дополнительных расходов (единоразовые траты) — учитываются пропорционально времени печати,
    - Ставки оператора (руб/час) и рабочих часов в день,
    - Автоматическое сохранение данных в LocalStorage,
    - Импорт/экспорт данных в формате JSON.

- **[Вторая версия](https://nazbav.github.io/3d-price/index2.html)** (index2.html)
  - Расширенный функционал. К уже имеющимся возможностям добавлен учёт:
    - **Мощности принтера (Вт)** и цены 1 кВт⋅ч для расчёта затрат на электроэнергию,
    - **Износа/обслуживания** принтера (руб/час),
    - **Процента брака (wastePercent)** для каждого материала,
    - **Упаковки/доставки** (shipping),
    - **Налогов** (taxPercent),
    - Более детализированный вывод разбивки расходов (амортизация, оператор, материал, электричество, обслуживание, дополнительная сумма, налог).

Во второй версии также сохранена **полноценная CRUD-функциональность** для:
- Принтеров
- Материалов
- Доп. расходов

и расширенный блок настроек калькулятора (рабочий график, дата начала, цена кВт⋅ч, налог и т. д.).

## Как использовать

1. **Открыть страницу**:
   - Первая версия: [index.html](https://nazbav.github.io/3d-price/index.html)
   - Вторая версия: [index2.html](https://nazbav.github.io/3d-price/index2.html)

   Приложение сразу подгрузится в браузере.

2. **Добавить или отредактировать данные**:
   - Перейдите на вкладку «Принтеры», «Материалы» или «Доп. расходы».
   - Нажмите кнопку «Добавить», чтобы внести новую запись.
   - Нажмите «Редактировать», чтобы изменить существующую запись (окна `prompt`).
   - Нажмите «Удалить», чтобы удалить выбранную запись.

3. **Перейти во вкладку «Калькулятор»**:
   - Выберите нужный принтер и материал,
   - Укажите время печати, вес изделия, ставку оператора, и другие параметры (рабочие часы, дата начала, цена кВт⋅ч и т.д.).
   - Нажмите «Рассчитать», чтобы увидеть итоговую стоимость печати и подробную разбивку расходов.

4. **Импорт/Экспорт JSON**:
   - Во вкладке «Импорт/Экспорт JSON» можно выгрузить текущее состояние таблиц (принтеры, материалы, доп. расходы, настройки) в текстовый файл или буфер обмена.
   - Можно импортировать ранее сохранённый или отредактированный JSON обратно.

5. **LocalStorage**:
   - При каждом изменении (добавлении/редактировании/удалении строк) данные автоматически сохраняются в LocalStorage вашего браузера.
   - При обновлении страницы все введённые данные остаются (до очистки кэша или ручного сброса).
